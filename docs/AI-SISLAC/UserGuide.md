# Guia do Usuário — Assistente SISLAC

O Assistente SISLAC é o atalho conversacional do sistema. Acesse pelo ícone flutuante no canto inferior direito de qualquer tela.

## Modos

- **Texto** — digite no campo e pressione Enter.
- **Voz** — toque no ícone do microfone, fale, toque novamente para enviar.

## Exemplos de comandos

| Você diz                                              | O que acontece |
|-------------------------------------------------------|----------------|
| "Abrir atendimentos"                                  | Navega para `/atendimentos`. |
| "Pesquisar paciente Maria Silva"                      | Lista pacientes que batem com o nome. |
| "Quantos atendimentos esta semana?"                   | Contagem agregada. |
| "Abrir atendimento 0000003"                           | Abre `/resultado/{id}` direto. |
| "Hemácias 4,5 e hemoglobina 13,8" (com exame aberto)  | Grava ambos os parâmetros após confirmação. |
| "Salvar"                                              | Confirma a gravação pendente. |

## Confirmações

Ações que alteram dados (cadastrar paciente, gravar resultado) exibem um diálogo "Confirmar?" antes de executar. Você sempre tem a palavra final.
